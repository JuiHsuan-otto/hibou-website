A

新增部落客推薦文章

觸發條件：使用者提供部落客名稱、標題、內容、推薦商品

1
決定文章 slug（檔名）
命名規則：article-[品牌縮寫]-[商品縮寫]-[流水號].html
範例：article-adoufu-windcover-001.html、article-boli-blanket-001.html
2
複製 article-template.html，替換所有 {{變數}}
變數說明範例
{{ARTICLE_TITLE}}完整文章標題喜福育兒好物分享｜寶寶我最罩
{{ARTICLE_META_DESC}}160字內 SEO 摘要阿呆夫妻使用喜福防風罩…
{{BLOGGER_HANDLE}}部落客名稱（含書名號）《阿呆夫妻過日子》
{{PRODUCT_TAG_1/2}}推薦商品標籤機能防風罩
{{BLOGGER_QUOTE}}部落客金句（引言框）超必買！用了四年還是最愛
{{PRODUCT_NAME}}商品名稱機能防風罩
{{TAG_1~3}}文章標籤防風罩
3
在 news-bloggers.html 對應分類的 .reviews-grid 內新增卡片 
<a href="article-[slug].html" class="review-list-card reveal" data-cat="[category]">

  <div class="rlc-title">文章標題</div>
  <div class="rlc-tags">
    <span class="rlc-tag">商品標籤</span>
  </div>
</a>

四、品牌聲音規範

語氣原則
  • ✅ 溫暖、有媽媽溫度，不冷硬
  • ✅ 品牌自稱「喜福」或「我們」
  • ✅ 對消費者用「媽媽」、「您」
  • ❌ 不用「消費者」「用戶」等詞
品牌核心語句
  • 「最私心的呵護」
  • 「為守護孩子而幸福誕生」
  • 「給孩子的永遠都要最好的」
  • 「MIT 台灣製造・SGS 嚴格把關」
商品描述必帶認證（擇一以上）
  • ✅ SGS-CNS 15290 國家標準
  • ✅ BSMI M55352 商品安全標章
  • ✅ MIT 台灣製造
  • ✅ 日本三河木棉認證（六重紗）
禁止使用詞彙
禁用改用
最好極致、嚴格把關
第一名媽媽首選、熱銷
保證通過檢驗、品質把關

五、常見問題排除

Q1：reveal 動畫元素不出現

確認新增元素有加 .reveal class，且 JS 已執行 observer.observe(el)。JS 在頁面底部執行，新增元素後頁面需重載。

Q2：行動版選單點擊沒反應

確認每頁只有一個 id="hamburger" 和 id="mobileNav"，不可重複。

Q3：Filter 篩選後卡片消失不掉

data-cat 屬性值拼錯。對照 Section 3 商品分類對照表，確認全小寫英文、無空格。

Q4：數字動畫不觸發

.stat-number 的直接文字節點只能含數字，<sup> 等標籤不影響，但 + 或 % 等符號放在 <sup> 內才能正確解析。

Q5：goPage() 找不到目標頁面

在 index.html JS 的 goPage() 函式內，補充對應的 key-value 路由。

六、發布前 Checklist

每次更新完成後，點擊勾選確認所有項目再發布。

  • <title> 已更新，包含關鍵字
  • <meta description> 已填寫（非空、非佔位符，120-160字）
  • 所有 {{...}} 佔位符已替換完畢
  • 圖片 src 路徑正確（以 ./images/ 開頭）
  • 所有圖片 alt 文字已填寫
  • 新文章已加入 news-bloggers.html 列表
  • 跑馬燈已同步更新（正本 + 複製本）
  • 行動版選單可正常開關
  • reveal 動畫可在捲動時觸發
  • Footer 連結正確（不指向空 #）
  • Breadcrumb 路徑正確
  • 品牌認證資訊至少包含一項
